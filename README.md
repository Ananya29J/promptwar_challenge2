🗳️ VoteWise: AI-Powered Election Guide Assistant
VoteWise is a high-fidelity, interactive web application designed to transform complex electoral processes into an engaging, gamified, and personalized experience. Built with a focus on civic education and voter empowerment, it utilizes AI to provide tailored guidance for every citizen.

🚀 Key Features
👤 Digital Twin Profile: An AI-driven wizard that analyzes your voter profile (age, awareness, location) to generate a personalized "Election Journey" checklist and dashboard.
🎮 Interactive Simulator: A gamified "Election Game" where users navigate real-world voting scenarios, learning the consequences of their choices in a safe, simulated environment.
🔍 Know Your Candidates (Dual Mode):
Ongoing Mode: Explore candidate backgrounds, track records, assets, and campaign promises.
Completed Mode: View election results with an AI-highlighted winner, a summary of the losing candidates, and a "What They Are Doing Now" section for the elected representative.
🔮 "What-If" Scenario Engine: An AI-powered engine that analyzes hypothetical election situations (e.g., "What if I lose my voter ID?") and provides immediate, short-term, and legal consequences.
🗺️ Interactive Flow Map: A visual, step-by-step guide to the entire Indian election journey, from registration to the final announcement of results.
🤖 AI Chat Assistant: A persistent, context-aware chatbot trained on Indian electoral laws and procedures to provide instant answers to voter queries.
🎨 Design & UX
Aesthetic: Professional "Light Mode" design system inspired by the Indian flag (Saffron, White, and Green).
High Fidelity: Uses glassmorphism, smooth Framer Motion animations, and a high-contrast component system for maximum accessibility and visual appeal.
Responsive: Fully optimized for mobile, tablet, and desktop viewing.
🛠️ Tech Stack
Frontend: Next.js (App Router), React, TypeScript.
Styling: Tailwind CSS + Vanilla CSS Variables for a robust design system.
Animations: Framer Motion (for physics-based transitions and interactive elements).
AI Engine: Google Gemini 1.5 Flash (via @google/generative-ai).
Icons: Lucide-React.
